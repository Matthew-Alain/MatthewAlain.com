[Android (Java)] - Martial Arts Staff App

An android mobile app designed for the staff at the martial arts academy I worked at to record their work hours. This project involved working closely with my employer to create an app that would properly fit the business' needs, and could be practically utilized by the staff.

[Python (TKinter)] - Battleship Game with GUI

A recreation of the board game Battleship in Python. Created to practice using TKinter to create a GUI for my first individual graphics-based project.

[QA] - Connect-4 Quality Test Report

A quality testing report for a peer's recreation of the board game Connect-4. The report identifies the 40 cases tested, the results and conclusions derived from those tests, testing methodologies used, and how those methodologies resulted in the creation of new test cases. The tested program was entirely developed by another programmer.

[PHP] - Looking For Group Forum

A PHP-based web forum where indie video game creators make postings looking for other team members to round out their missing skillsets. This project was a collaborative effort alongside three additional programmers, with my role being the project manager and supporting back-end developer. I was responsible for all functionalities relating to creating, viewing, and storing posts in the database.

[Python (Streamlit)] - Leave Management System

A leave management system where users can create accounts with either manager or employee permissions, and then create leave requests for those accounts that their respective manager may approve or deny. This project was developed during a timed three-hour sprint with no functional requirements provided in advance, in a team with three other programmers. My role was creating the logical flow between each page, and establishing the connection between the site and the SQLite database.

[Processing 4] - Tree Planting Simulator

A simple game in which the player can move between screens to gather different tools to plant, water, and chop trees while protecting them from flies. Tests multitasking and time management. This project was a collaborative effort alongside two additional programmers, with my role being the refining of gameplay mechanics and game state transitions.
